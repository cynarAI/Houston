- **Tone**: Friendly, encouraging ("Bist du bereit?")
- **Language**: German, du-form (informal)
- **Messaging**: Empowering, mission-oriented
- **Keywords**: Mission, KI, Marketing, Strategie, Tools

## Key Branding Elements for Houston App

### Must-Have:
1. **Astronaut Mascot** - Houston as astronaut character
2. **Cyan/Purple/Pink Gradient** - Primary brand colors
3. **Space Theme** - Mission terminology, space metaphors
4. **Clean, Modern Design** - Lots of whitespace
5. **Friendly Tone** - Encouraging, empowering

### Color Application:
- **Landing Page**: Keep space theme (dark bg, stars) but add AIstronaut colors
- **App**: Light mode with cyan/purple accents, gradient CTAs
- **Buttons**: Pink-purple-blue gradients
- **Highlights**: Cyan for important elements

### Typography:
- Use modern sans-serif (Inter or similar)
- Bold headings
- Clear hierarchy

### Imagery:
- Add astronaut illustrations/icons
- Space-themed graphics
- Mission/rocket metaphors
